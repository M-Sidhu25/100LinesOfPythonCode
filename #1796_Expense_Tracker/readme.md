```Expense Tracker CLI```

A simple **command-line expense tracker** built in Python to help users record, view, and analyze their personal expenses.  

---

## Features

- **Add Expense:** Record an expense with **date**, **amount**, and **category**.
- **View Expenses:** Display all recorded expenses in a readable format.
- **Print Summary:** Show **category-wise totals** to track spending.
- **Check Overspending:** Compare category totals against a **user-defined limit**.
- **Menu-Driven CLI:** Easy-to-use menu interface for repeated actions.
- **Robust Input Validation:** Ensures valid dates, positive numeric amounts, and handles errors gracefully.
