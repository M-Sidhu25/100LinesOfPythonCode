# Minimal Tetris Game 🎮 (Pygame)

A **compact Tetris game** built in Python with Pygame, written **under 100 lines**.

## How to Play
- Arrow Keys:
  - ⬅️ Left
  - ➡️ Right
  - ⬇️ Down
  - ⬆️ Rotate
- Fill rows to clear them.
- Game ends if blocks reach the top.

## Requirements
- Python 3.x
- Pygame library

pip install pygame
