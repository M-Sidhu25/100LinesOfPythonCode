# Sudoku Solver & Generator

This script generates a **random Sudoku puzzle** and solves it using **backtracking** — all in under 100 lines of Python.

### 🔹 Features
- Generates a valid Sudoku puzzle
- Solves puzzles using backtracking
- Clean, readable 9x9 grid output
- Easy to modify (e.g., change difficulty)

### ⚙️ How to Run
```bash
python sudoku_solver_generator.py
